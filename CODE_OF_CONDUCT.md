# Code of Conduct

## Our Pledge

We pledge to make participation in DjimFlo a harassment-free experience for everyone.

## Our Standards

- Be respectful and inclusive
- Accept constructive criticism gracefully
- Focus on what is best for the community and the software
- Show empathy towards others

## Unacceptable Behavior

- Harassment, discrimination, or derogatory language
- Trolling, personal attacks, or political warfare
- Publishing others' private information without permission
- Anything that would make a reasonable person uncomfortable

## Enforcement

Violations may result in:
1. Warning
2. Temporary ban from contributions
3. Permanent ban

Report violations to: dlandman@djimit.nl

## Scope

This code of conduct applies within all project spaces and when representing the project.
